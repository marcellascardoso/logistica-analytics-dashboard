# 🚚 Dashboard de Logística – Analytics & Performance

Este projeto tem como objetivo analisar dados logísticos de transporte rodoviário, utilizando indicadores de custos, prazos e volumes para apoiar a tomada de decisão operacional e estratégica.

O dashboard foi desenvolvido no **Power BI**, com base em dados históricos de transporte extraídos de arquivos **Excel** e **TXT**, permitindo uma visão completa da performance logística.

---

## 🎯 Objetivos do Projeto

- Monitorar o **custo total de frete**
- Avaliar o indicador **On Time Delivery (OTD)**
- Analisar **volume transportado** (peso e cubagem)
- Comparar performance por **região (UF)**
- Avaliar utilização por **tipo de veículo**
- Apoiar decisões relacionadas à **eficiência logística**

---

## 📊 Indicadores (KPIs)

- **Custo Total de Transporte**
- **Total de Viagens**
- **Entregas Dentro do Prazo**
- **On Time Delivery (OTD)**
- **Peso Bruto Transportado (Kg)**
- **Cubagem Total (m³)**

---

## 📈 Visuais do Dashboard

- Valor do frete ao longo do tempo
- Indicador de OTD com meta
- Viagens por tipo de veículo
- Performance de entregas por região
- Mapa geográfico de distribuição das entregas
- Cards com KPIs consolidados

---

## 🗂️ Fonte de Dados

- **Historico_Transporte.xlsx**
  - Informações de viagens, prazos, origem/destino, peso e cubagem
- **Valor_do_Frete.txt**
  - Valores de frete associados às viagens

Os dados representam registros históricos simulados para fins de estudo e análise.
